# scala-cli-maven-surefire-spring-findbugs-pmd-cucumber-mockito-junit-car-data

## Description
A POC for maven app using junit5
and cucumber and mockio framework
 with findbugs,
pmd and urefire plugin.

## Tech stack
- scala
- maven
	- spring
	- mockito
  - junit5
  - findbugs
  - pmd
  - surefire
  - cucumber

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
